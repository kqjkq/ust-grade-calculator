# Grade Calculator Web App

A modern, responsive grade calculator that helps students track their academic progress and plan their study efforts effectively.

## Features

- Calculate current grade based on completed assignments/exams
- Determine scores needed on remaining work to reach target grades
- Personalized study plan recommendations
- Responsive design that works on all devices
- No backend required - runs entirely in the browser

## How to Use

1. Clone or download this repository
2. Open `index.html` in a web browser
3. Select your course and professor
4. Enter your scores and target grade
5. View your current grade and study recommendations

## Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome for icons
- Google Fonts (Inter)
- No external dependencies required

## Customization

To add your own courses or modify the grading structure, edit the `js/data/courses.js` file.
